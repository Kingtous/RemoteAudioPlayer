### Remote Audio Player

Play Audio on your Linux devices using your phone.

**Requirements**

- Python Packages
  - pyaudio
  - pybluez
  - ffmpeg
  - pydub

**What you need to do**

- Download the [Release](https://github.com/Kingtous/RemoteAudioPlayer-Android/releases) App.
- Running `__init__.py` on your remote devices.
  - Linux Required
